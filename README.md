# Gradient Generator

Create beautiful CSS gradients with live preview and copy-ready code.

## How to Use

**Step 1:** Open `index.html` in your browser.

**Step 2:** An onboarding tutorial appears on first visit. Click **Start Creating** (or Skip).

**Step 3:** Click the color pickers to choose 2-3 colors for your gradient.

**Step 4:** Drag the **Angle** slider to change the gradient direction (0-360 degrees).

**Step 5:** Switch between **Linear** and **Radial** gradient types.

**Step 6:** The preview updates in real-time as you change settings.

**Step 7:** The CSS code appears below. Click **Copy CSS** to copy it to clipboard.

**Step 8:** Click any **preset** at the bottom to quickly apply a pre-made gradient.

## Features
- Live gradient preview
- 3 color stops
- Angle control (0-360)
- Linear and Radial modes
- One-click CSS copy
- 12 preset gradients
- Real-time updates

## Tech
HTML + CSS + JavaScript | Clipboard API
